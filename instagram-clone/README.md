<h2>Mini 9Gag</h2>
<p>A minimilistic version of 9gag built with React and Firebase</p>
<br> <br>

<h2> Getting started </h2>

<h3>Installation</h3>
-Install Node.js + React
<br> <br>

- Add firebase to your project
Use your firebase crendentials and store it in -src/firebase.js
<img src ="./screenshots/firebase.png" width="100%" height="100%"/>

<br>
- git clone https://github.com/Raashul/CU-CS-club.git

cd instagram-clone

npm install

<br> <br>

<h3>Running the app</h3>

npm start
